# SQL Study

> 연이 끊기지 않는 이상 계속 됩니다.

## ⚖️ 스터디 규칙
- 1주일 1회 만남을 가지면서, 학습내용 공유 및 SQL 문제풀이 공유
- 언어 : SQL
- SQL 문제 : [HackerRank](https://www.hackerrank.com/)
- 참고 사이트 : [왕초보용! 갖고 노는 MySQL 데이터베이스 강좌](https://www.youtube.com/watch?v=dgpBXNa9vJc&t=8460s)

- SQL 문법을 터득한다.
- 문제를 풀고 코드리뷰 진행
